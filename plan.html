一、背景与价值

夜光杯实物图：采用祁连山玉石制作，工艺精湛

背景

西周时期（公元前七世纪）

西域小国向周穆王进贡夜光常满杯，成为夜光杯的起源

唐代

诗人王翰《凉州词》中"葡萄美酒夜光杯"名句使夜光杯名扬天下

明末清初

夜光杯生产进入兴盛期，酒泉有十多家玉石作坊制作夜光杯

2006年

酒泉夜光杯雕被列入第一批国家级非物质文化遗产代表性项目名录

2016年

"酒泉夜光杯"被批准为国家地理标志产品

价值

历史价值

见证了丝绸之路的商贸往来和文化交流

艺术价值

融合了玉石工艺与饮酒文化，体现了中国传统美学

工艺价值

38道复杂工序，展现了传统手工艺的精湛技艺

文化价值

承载了河西走廊的历史记忆和文化传承

经济价值

成为当地特色产业，促进文化旅游发展

二、数据来源与处理方案

数据可视化示例：展示夜光杯产业相关数据

数据来源

1. 文化遗产数据

• 中国非物质文化遗产网（项目编号：Ⅶ-31）：包含夜光杯雕的详细申报资料、保护计划和传承情况

• 国家地理标志产品保护信息：夜光杯的地理标志认证文件、质量标准和生产规范

• 甘肃省文化和旅游厅官方资料：关于夜光杯的保护政策、产业发展规划和文化推广活动

• 酒泉市人民政府官方网站：夜光杯产业发展数据和政策支持措施

2. 历史文献数据

• 《十洲记》《穆天子传》等古代文献：关于夜光杯起源的历史记载

• 唐代诗歌及历史典籍：如王翰《凉州词》等提及夜光杯的文学作品

• 地方史志资料：《酒泉府志》《肃州志》等地方文献中关于夜光杯的记载

• 考古发掘报告：河西走廊地区与夜光杯相关的考古发现

3. 工艺数据

• 酒泉夜光杯厂工艺流程资料：38道制作工序的详细说明和技术参数

• 非遗传承人口述历史：工艺传承故事、技术秘诀和经验总结

• 制作工序视频资料：从选料到成品的完整制作过程记录

• 工艺工具和设备数据：传统工具的规格、使用方法和现代设备的应用

4. 现代数据

• 生产数据：2023年年产量12万件，30%出口至"一带一路"沿线国家

• 市场销售数据：国内销售网络分布、国际市场拓展情况和销售额统计

• 用户反馈数据：消费者对不同系列产品的评价和需求分析

• 文创产品开发情况：近年来开发的新款式、新用途和市场反应

• 旅游数据：夜光杯作为旅游纪念品的销售情况和游客反馈

处理方案

1. 数据收集

  • 官方渠道：通过政府网站、非遗保护中心等官方机构获取权威数据
  • 实地调研：前往酒泉夜光杯厂进行实地考察，拍摄制作过程
  • 文献查阅：搜集相关历史文献和学术研究资料
  • 专家访谈：与非遗传承人、工艺大师进行深度访谈
  • 市场调研：收集市场销售数据和用户反馈

2. 数据整理

  • 分类归档：将数据按历史、工艺、市场等类别进行分类
  • 数据清洗：去除重复和无效数据，确保数据质量
  • 验证核实：通过多方渠道验证数据的真实性和准确性
  • 标准化处理：统一数据格式和计量单位，便于后续分析

3. 数据可视化

  • 时间序列数据：使用折线图展示产量、销售额等随时间的变化
  • 地理分布数据：使用地图可视化展示原料产地、销售网络等
  • 工艺流程数据：使用流程图展示制作工序的先后顺序和关系
  • 市场分析数据：使用饼图、柱状图展示市场份额和用户偏好
  • 三维数据：使用3D模型展示夜光杯的造型和结构

4. 数据更新

  • 建立数据采集机制：定期从官方渠道获取最新数据
  • 设置数据更新周期：根据数据类型确定更新频率
  • 建立数据备份系统：确保数据安全和可追溯性
  • 开发数据管理界面：方便团队成员查看和更新数据

三、可视化设计思路

3D模型展示：交互式查看夜光杯的造型和细节

1. 历史时间轴

设计目标：通过交互式时间轴直观展示夜光杯的发展历程，突出重要历史节点和文化事件

技术实现：使用HTML5 Canvas或SVG绘制时间轴，结合JavaScript实现交互效果

功能特点：

  • 可拖动时间轴，查看不同历史时期
  • 点击时间节点显示详细历史事件
  • 融入历史图片和文献资料
  • 支持时间轴缩放功能

预期效果：用户可以清晰了解夜光杯从西周至今的发展脉络，感受其历史文化价值

2. 工艺流程

设计目标：动态展示夜光杯38道制作工序，突出传统工艺的复杂性和精湛技艺

技术实现：使用CSS动画和JavaScript实现工序流程的动态展示

功能特点：

  • 分阶段展示制作工序，每道工序配有详细说明
  • 融入制作过程的视频片段
  • 支持工序间的跳转和回溯
  • 展示关键工序的技术要点

预期效果：用户可以直观了解夜光杯的制作过程，感受传统工艺的魅力

3. 3D模型展示

设计目标：构建夜光杯的高精度3D模型，支持多角度查看，展示其造型和纹理细节

技术实现：使用Three.js或Babylon.js构建3D模型，实现交互控制

功能特点：

  • 支持360°旋转查看
  • 可缩放模型，查看细节
  • 展示不同种类夜光杯的3D模型
  • 融入材质和光影效果，增强真实感

预期效果：用户可以全方位了解夜光杯的造型设计和工艺细节

4. 文化地图

设计目标：基于GIS技术，展示夜光杯的原料产地、制作中心和销售网络，呈现其文化传播路线

技术实现：使用Leaflet或Mapbox等地图库，结合自定义图层

功能特点：

  • 标记祁连山玉矿产地
  • 展示夜光杯制作中心分布
  • 呈现销售网络和文化传播路线
  • 支持地图缩放和区域筛选

预期效果：用户可以直观了解夜光杯的地理分布和文化传播范围

5. 数据仪表盘

设计目标：展示夜光杯的生产、销售和市场数据，支持交互式查询和分析

技术实现：使用Chart.js、ECharts等图表库，结合前端框架实现

功能特点：

  • 产量趋势图表
  • 市场分布饼图
  • 销售数据柱状图
  • 支持数据筛选和时间范围选择

预期效果：用户可以通过数据直观了解夜光杯产业的发展状况

6. 虚拟现实体验

设计目标：通过VR技术，让用户沉浸式体验夜光杯的制作过程和文化内涵

技术实现：使用A-Frame或Three.js VR模块，结合WebVR API

功能特点：

  • 虚拟参观夜光杯制作车间
  • 模拟体验部分制作工序
  • 360°全景展示制作环境
  • 融入音频解说和背景音乐

预期效果：用户可以获得身临其境的体验，深入了解夜光杯的制作工艺和文化背景

技术实现

前端技术栈

• HTML5：构建页面结构和语义化标签

• CSS3：实现响应式布局和视觉效果

• JavaScript：实现交互功能和数据处理

• Three.js：构建3D模型和VR体验

• Chart.js/ECharts：实现数据可视化图表

• Leaflet/Mapbox：实现地理信息可视化

设计原则

• 移动优先：确保在手机、平板等移动设备上的良好体验

• 交互友好：设计直观的用户界面和交互方式

• 视觉统一：保持整体设计风格的一致性

• 性能优化：确保页面加载速度和交互响应速度

• 可访问性：确保不同用户群体都能正常使用

实现步骤

1. 需求分析：确定核心功能和用户需求

2. 原型设计：制作页面原型和交互流程图

3. 数据准备：收集和整理所需数据

4. 前端开发：实现页面结构和基本样式

5. 功能开发：实现各个可视化模块和交互功能

6. 测试优化：进行功能测试和性能优化

7. 部署发布：准备竞赛提交材料

四、同类可参考作品（重点分析）

参考作品展示：传统工艺数字化可视化案例

1. 《解构天章-中国古代天文学大数据视觉图谱》（呼伦贝尔学院）

获奖情况：中国大学生计算机设计大赛获奖作品

作品内容：融合科学与艺术的可视化作品，以中国古代天文学智慧为基础，结合当代国家空间科学发展战略进行设计与创作。

设计特点： • 四大模块：人物著作篇、天文仪器篇、行星篇、天文历法篇
• 信息呈现：通过信息图形、数据对比与动效叙事，直观展现中国天文学在历史与未来的传承与创新
• 技术实现：引入真实的天体比例与运转数据，制作八大行星与太阳的等比例动画，以及浑天仪等古代仪器的动态演示
• 视觉效果：所有素材均为原创，部分参考古籍文献进行高还原度绘制，既体现历史的厚重感，又展现现代设计的审美与科技感

参考价值：多模块信息架构设计和动效叙事的思路，可应用于夜光杯的历史文化展示

2. 《汝瓷制作工艺可视化设计》（广州城市理工学院）

获奖情况：中国大学生计算机设计大赛全国一等奖

作品内容：运用现代信息可视化技术，整合汝瓷历史、工艺、艺术、文化等多维度数据，通过精妙交互设计、清晰数据图表和东方美学视觉呈现，多维度展现汝瓷魅力。

设计特点： • 工艺展示：涵盖开采、淘石等十大制作工序，解析胎釉热胀冷缩差异形成蝉翼纹等独特开片的原理
• 视觉呈现：以天青色为主的釉色、多样开片纹理及瓶、杯托等丰富器型的可视化展示
• 交互设计：通过精妙的交互设计，让用户可以探索汝瓷的制作过程和文化内涵
• 数据整合：整合历史、工艺、艺术、文化等多维度数据，构建完整的汝瓷知识体系

参考价值：其多维度数据整合和交互设计思路可应用于夜光杯项目

3. 《凭起落，定风波》（华东师范大学）

获奖情况：中国大学生计算机设计大赛全国一等奖

作品内容：以《定风波·莫听穿林打叶声》为教学内容，将“以美育人”理念与数字化技术深度融合，针对初中生由形象思维向抽象思维过渡的认知特点，引导学生深入体悟古诗词之美。

设计特点： • 教学内容：以《定风波·莫听穿林打叶声》为核心，结合新课标中“文化自信”核心素养
• 技术应用：运用即梦、可灵、Suno、Runway、Vidu等六类AIGC工具，丰富表现形式
• 学习体验：为学生带来全新的学习体验，帮助学生理解古诗词中的文化基因与精神标识
• 教育理念：将“以美育人”理念与数字化技术深度融合，实现“以文化人”的教育目标

参考价值：传统文化与现代技术融合的设计思路，可应用于夜光杯文化内涵的展示

4. 《沙海遗珍 -- 敦煌文化数字转译》（黄山学院）

获奖情况：中国大学生计算机设计大赛国赛三等奖

作品内容：以国际生“汉学”信息可视化设计作品形式，通过数字化手段展现敦煌文化的历史价值和艺术魅力，实现传统文化的现代转译。

设计特点： • 文化转译：通过数字化手段将敦煌文化的历史价值和艺术魅力转化为现代可理解的形式
• 国际视角：以国际生“汉学”作品形式，面向国际受众传播敦煌文化
• 信息可视化：运用信息可视化技术，清晰呈现敦煌文化的历史脉络和艺术特色
• 现代传播：通过现代数字技术，使传统文化在当代得到更广泛的传播

参考价值：传统文化数字化转译的设计思路，可应用于夜光杯文化的现代传播
